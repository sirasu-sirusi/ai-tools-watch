# AIツール料金ウォッチ — 運用計画

非属人型の収益化プロジェクト。独自データ(AIツール料金の週次追跡)を核に、note・YouTube・無料ダッシュボードの3面で展開する。

## 構成

- `index.html` — 比較ダッシュボード(GitHub Pages で無料公開予定)
- `data/tools.json` — 料金データベース(これが一次資産。毎週更新)
- `note記事_第1弾_下書き.md` — note初回記事の下書き

## 週次運用ループ(Claudeが実行)

1. 主要ツールの料金・プラン変更をWeb検索で確認
2. `tools.json` を更新(updated日付も)
3. 変更点をまとめた note 記事下書きを生成
4. (軌道に乗ったら)同データでYouTube用台本+グラフ生成

## 公開手順(初回のみ・要ユーザー操作)

1. GitHubアカウントで新規リポジトリ `ai-tools-watch` を作成(無料)
2. このフォルダをpush → Settings > Pages で公開
3. noteアカウントで下書きを投稿(無料部分+有料ライン設定300円)
4. ダッシュボードのnoteリンク(`index.html`の`note-link`)を実URLに差し替え

## 収益導線

- note有料記事(300円)/ 月額マガジン(500円)← 最初の収益源
- ダッシュボード→note への誘導
- 中期:YouTubeデータ可視化動画、アフィリエイト(各ツール紹介プログラム)
